Attack Methods

Common techniques used to compromise systems.

Phishing

Deceptive emails or messages designed to trick users into revealing sensitive information like passwords or credit card numbers.

See More

DDoS Attack

Distributed Denial of Service. Overwhelming a target server or network with a flood of internet traffic to disrupt normal operations.

See More

SQL Injection

Inserting malicious SQL code into input fields to manipulate a database, often allowing attackers to view or delete data.

See More

Ransomware

Malware that encrypts a victim's files. The attacker then demands a ransom from the victim to restore access to the data.

See More

Man-in-the-Middle

An attack where the attacker secretly relays and possibly alters the communications between two parties who believe they are directly communicating.

See More

Social Engineering

Psychological manipulation of people into performing actions or divulging confidential information.
